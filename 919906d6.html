前言

  • 作为一个非计算机专业的学生，搭载个人博客纯属个人兴趣使然。 (计算机专业也不是都有博客好吗！) 抱着网上教程丰富的想法，就开始着手搭载个人博客。结果上手之后发现，因为时间久远，遗留下来的各式各类的问题千奇百怪，为了记录一下解决这些问题的过程，我决定写下这篇教程。希望我的文章可以让读者一步步跟随直至成功搭载，让每个人都可以拥有自己的个人博客！ 顺便挑战一下自己的教学能力

按 Ctrl+C 关闭本地服务器。

以后我们常用的 Hexo 命令

  • hexo s 等价于 hexo server #Hexo 会监视文件变动并自动更新，除修改站点配置文件外，无须重启服务器，直接刷新网页即可生效。

  • hexo g 等价于 hexo generate #生成静态网页 (执行 $ hexo g 后会在站点根目录下生成 public 文件夹，hexo 会将 /Hexo/source/ 下面的 .md 后缀的文件编译为.html 后缀的文件，存放在 /Hexo/public/ 路径下)

  • hexo d 等价于 hexo deploy #将本地数据部署到远端服务器 (如 github)

  • hexo clean #清除缓存，网页正常情况下可以忽略此条命令，执行该指令后，会删掉站点根目录下的 public 文件夹

可以直接使用的指令串
hexo clean && hexo g && hexo s #快速本地预览
hexo clean && hexo g && hexo d #快速部署远端服务器

  • enable：默认为 true。在右下角显示返回顶部图标
  • sidebar：默认为 false。在侧边栏显示
